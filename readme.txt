# DeliverSync
DeliverSync is a microservice designed to optimize the process of assigning orders to riders based on their real-time location and availability, ensuring timely and efficient deliveries.

## Key Features
- Real-Time Updates
- Efficient Matching
- Scalability

## Tech Stack
- Node.js
- TypeScript
- Express.js
- MongoDB
- Socket.io
- Swagger
- Jest
